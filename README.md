# カーネルキーリング・マネージャー

カーネルキーリングのエントリをGUIで管理するためのGTK4アプリケーションです。

## 概要

このアプリケーションは、Linuxカーネルのキーリング機能（`keyutils`）を利用して、ユーザーのクレデンシャル（パスワードや秘密鍵など）を安全に管理するためのグラフィカルユーザーインターフェース（GUI）を提供します。パスワードなどの機密情報をファイルシステムに直接書き込むことなく、メモリ上で安全に保持できるため、セキュリティの向上に貢献します。

## 機能

-   ユーザーキーリング（`@u`）内のキーの一覧表示
-   新しいキーの追加
-   既存のキーの値の更新
-   キーの削除
-   単一インスタンス実行（2度目の起動で既存ウィンドウを前面に表示）
-   完全に日本語化されたUI

## インストール

### 必要条件

このアプリケーションをインストールして実行するには、以下のものが必要です。

-   Python 3.x
-   GTK4 (通常、`dev-python/pygobject` パッケージに含まれます)
-   `keyutils` コマンド (システムパッケージとしてインストールされている必要があります)

以下のPythonライブラリが必要です。

- pygobject
- keyutils

### アプリケーションのインストール

プロジェクトのルートディレクトリで `make install` コマンドを実行します。これにより、実行ファイル、アイコン、デスクトップエントリが `~/.local/` 以下にインストールされます。

```bash
make install
```

## 使い方

インストール後、お使いのデスクトップ環境のアプリケーションランチャーから「カーネルキーリング・マネージャー」を検索して起動できます。または、ターミナルから以下のコマンドで直接起動することも可能です。

```bash
keyring-manager
```

## アンインストール

アプリケーションを削除するには、プロジェクトのルートディレクトリで `make uninstall` コマンドを実行します。

```bash
make uninstall
```
